# Library Management System

## Project Description
This project is a Library Management System developed using HTML, CSS, and JavaScript. It helps users search for books, check availability, view book details, and manage books.

## Features
- Search Books
- View Availability
- Book Details
- Category Filter
- Admin Management

## Technologies Used
- HTML
- CSS
- JavaScript
- Git
- GitHub
